## Overview

Retrieves complete details of a specific recommendation including user, terrain, tractor, implement data, compatibility score, and detailed observations.

<Info>
  Users can only access their own recommendations.
</Info>

## Authentication

**Required:** Bearer token in Authorization header

```bash
Authorization: Bearer <your_jwt_token>
```

## Path Parameters

<ParamField path="id" type="integer" required>
  The unique identifier of the recommendation
</ParamField>

## Examples

<CodeGroup>
```bash cURL
curl -X GET "http://localhost:4000/api/recommendations/15" \
  -H "Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..." \
  -H "Content-Type: application/json"
```

```javascript JavaScript
const recommendationId = 15;

const response = await fetch(
  `http://localhost:4000/api/recommendations/${recommendationId}`,
  {
    method: 'GET',
    headers: {
      'Authorization': `Bearer ${token}`,
      'Content-Type': 'application/json'
    }
  }
);

const data = await response.json();
const observations = JSON.parse(data.data.observations);
console.log(`Rank: ${observations.rank}, Score: ${data.data.compatibility_score}`);
```

```python Python
import requests
import json

token = "your_jwt_token_here"
recommendation_id = 15

response = requests.get(
    f'http://localhost:4000/api/recommendations/{recommendation_id}',
    headers={
        'Authorization': f'Bearer {token}',
        'Content-Type': 'application/json'
    }
)

data = response.json()['data']
observations = json.loads(data['observations'])
print(f"Tractor: {data['tractor_name']}")
print(f"Score: {data['compatibility_score']}/100")
print(f"Explanation: {observations['explanation']}")
```
</CodeGroup>

## Error Responses

| Status Code | Description |
|-------------|-------------|
| **401** | Missing or invalid authentication token |
| **404** | Recommendation not found or doesn't belong to user |
| **500** | Internal server error |

## Observations Field

The `observations` field contains a JSON string with detailed scoring information:

```json
{
  "rank": 1,
  "scores": {
    "efficiency": 28,
    "traction": 25,
    "soil": 18,
    "economic": 10,
    "availability": 10
  },
  "explanation": "Detailed reasoning for the recommendation"
}
```
