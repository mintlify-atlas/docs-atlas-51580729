## Overview

Retrieves the recommendation history for the authenticated user with pagination support. Results are cached for 1 hour.

<Info>
  This endpoint only returns recommendations generated by the authenticated user.
</Info>

## Authentication

**Required:** Bearer token in Authorization header

```bash
Authorization: Bearer <your_jwt_token>
```

## Query Parameters

<ParamField query="page" type="integer" default="1">
  Page number for pagination
</ParamField>

<ParamField query="limit" type="integer" default="10">
  Number of results per page (max 100)
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Indicates if the request was successful
</ResponseField>

<ResponseField name="data" type="array">
  Array of recommendation records
  
  <Expandable title="Recommendation object properties">
    <ResponseField name="recommendation_id" type="integer">
      Unique recommendation identifier
    </ResponseField>
    
    <ResponseField name="terrain_name" type="string">
      Name of the terrain
    </ResponseField>
    
    <ResponseField name="tractor_name" type="string">
      Name of the recommended tractor
    </ResponseField>
    
    <ResponseField name="implement_name" type="string">
      Name of the implement
    </ResponseField>
    
    <ResponseField name="compatibility_score" type="number">
      Overall compatibility score (0-100)
    </ResponseField>
    
    <ResponseField name="work_type" type="string">
      Type of agricultural work
    </ResponseField>
    
    <ResponseField name="recommendation_date" type="string">
      ISO 8601 timestamp of when the recommendation was generated
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="pagination" type="object">
  Pagination metadata
  
  <Expandable title="Pagination properties">
    <ResponseField name="currentPage" type="integer">
      Current page number
    </ResponseField>
    
    <ResponseField name="totalPages" type="integer">
      Total number of pages
    </ResponseField>
    
    <ResponseField name="totalItems" type="integer">
      Total number of recommendations
    </ResponseField>
    
    <ResponseField name="itemsPerPage" type="integer">
      Number of items per page
    </ResponseField>
  </Expandable>
</ResponseField>

## Examples

<CodeGroup>
```bash cURL
curl -X GET "http://localhost:4000/api/recommendations/history?page=1&limit=10" \
  -H "Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..." \
  -H "Content-Type: application/json"
```

```javascript JavaScript
const response = await fetch(
  'http://localhost:4000/api/recommendations/history?page=1&limit=10',
  {
    method: 'GET',
    headers: {
      'Authorization': `Bearer ${token}`,
      'Content-Type': 'application/json'
    }
  }
);

const data = await response.json();
console.log(`Total recommendations: ${data.pagination.totalItems}`);
```

```python Python
import requests

token = "your_jwt_token_here"

response = requests.get(
    'http://localhost:4000/api/recommendations/history',
    headers={
        'Authorization': f'Bearer {token}',
        'Content-Type': 'application/json'
    },
    params={'page': 1, 'limit': 10}
)

data = response.json()
for rec in data['data']:
    print(f"{rec['tractor_name']} for {rec['terrain_name']}: {rec['compatibility_score']}/100")
```
</CodeGroup>

## Error Responses

| Status Code | Description |
|-------------|-------------|
| **401** | Missing or invalid authentication token |
| **500** | Internal server error |

## Caching

<Note>
  Responses are cached for 1 hour (3600 seconds) to improve performance. Recent recommendations may take up to an hour to appear in the history.
</Note>
